### –§–æ—Ä–º—É–ª—ã –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤

**Explainability Score:**

```
ES = w_c √ó F_c + w_s √ó C_s + w_i √ó S_i + w_h √ó U_h

–≥–¥–µ:
F_c = |E_sys ‚à© E_exp|/|E_exp| √ó Œ± + |E_sys ‚à© E_exp|/|E_sys| √ó (1-Œ±)
C_s = 1 - H(E)/H_max
S_i = 1 - (1/N) √ó Œ£ d(E_i, E_i^Œµ)
```

**Robustness Index:**

```
RI = w_a √ó R_a + w_n √ó R_n + w_e √ó R_e

–≥–¥–µ:
R_a = 1 - (1/|A|) √ó Œ£ I[f(x + a) ‚â† f(x)]
R_n = E[similarity(f(x), f(x + Œµ))]
R_e = E[similarity(E(x), E(x + Œµ))]
```

**Concept-Drift Rate:**

```
CDR = Œª √ó KS(P_t, P_t-Œît) + (1-Œª) √ó JS(P_t, P_t-Œît)
```

**Bias Shift Index:**

```
BSI = ‚àö(w_dp √ó DP_Œî¬≤ + w_eo √ó EO_Œî¬≤ + w_cf √ó CF_Œî¬≤)
```
