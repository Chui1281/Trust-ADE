## üìä –ü—Ä–æ—Ç–æ–∫–æ–ª Trust-ADE

### üéØ –û—Å–Ω–æ–≤–Ω–∞—è —Ñ–æ—Ä–º—É–ª–∞

–ü—Ä–æ—Ç–æ–∫–æ–ª –∞–≥—Ä–µ–≥–∏—Ä—É–µ—Ç —Ç—Ä–∏ –∫–ª—é—á–µ–≤—ã—Ö –∏–∑–º–µ—Ä–µ–Ω–∏—è –≤ –µ–¥–∏–Ω—É—é –º–µ—Ç—Ä–∏–∫—É –¥–æ–≤–µ—Ä–∏—è:

```
Trust_ADE = w_E √ó ES + w_R √ó (RI √ó e^(-Œ≥ √ó CDR)) + w_F √ó (1 - BSI)
```

**–ì–¥–µ:**

- **ES** - Explainability Score (–æ—Ü–µ–Ω–∫–∞ –æ–±—ä—è—Å–Ω–∏–º–æ—Å—Ç–∏)
- **RI** - Robustness Index (–∏–Ω–¥–µ–∫—Å —É—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç–∏)
- **CDR** - Concept-Drift Rate (—Å–∫–æ—Ä–æ—Å—Ç—å –¥—Ä–µ–π—Ñ–∞ –∫–æ–Ω—Ü–µ–ø—Ç–æ–≤)
- **BSI** - Bias Shift Index (–∏–Ω–¥–µ–∫—Å —Å–º–µ—â–µ–Ω–∏—è –ø—Ä–µ–¥–≤–∑—è—Ç–æ—Å—Ç–∏)
- **w_E, w_R, w_F** - –¥–æ–º–µ–Ω–Ω–æ-—Å–ø–µ—Ü–∏—Ñ–∏—á–Ω—ã–µ –≤–µ—Å–∞
- **Œ≥** - –ø–∞—Ä–∞–º–µ—Ç—Ä —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∫ –¥—Ä–µ–π—Ñ—É –∫–æ–Ω—Ü–µ–ø—Ç–æ–≤


### üîç –ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –ø—Ä–æ—Ç–æ–∫–æ–ª–∞

#### 1. Explainability Score (ES)

–û—Ü–µ–Ω–∏–≤–∞–µ—Ç –∫–∞—á–µ—Å—Ç–≤–æ –æ–±—ä—è—Å–Ω–µ–Ω–∏–π —á–µ—Ä–µ–∑ —á–µ—Ç—ã—Ä–µ –∏–∑–º–µ—Ä–µ–Ω–∏—è:

```
ES = w_c √ó F_c + w_s √ó C_s + w_i √ó S_i + w_h √ó U_h
```

- **F_c** - –ö–∞—É–∑–∞–ª—å–Ω–∞—è —Ñ–∏–¥–µ–ª–∏—Ç–∏:

```
F_c = |E_sys ‚à© E_exp|/|E_exp| √ó Œ± + |E_sys ‚à© E_exp|/|E_sys| √ó (1-Œ±)
```

- **C_s** - –°–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∞—è –∫–æ–≥–µ—Ä–µ–Ω—Ç–Ω–æ—Å—Ç—å:

```

```

- **S_i** - –°—Ç–∞–±–∏–ª—å–Ω–æ—Å—Ç—å –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏–π:

```

```

- **U_h** - –ß–µ–ª–æ–≤–µ—á–µ—Å–∫–∞—è –ø–æ–Ω—è—Ç–Ω–æ—Å—Ç—å (—ç–∫—Å–ø–µ—Ä—Ç–Ω–∞—è –æ—Ü–µ–Ω–∫–∞)


#### 2. Robustness Index (RI)

–ò–Ω—Ç–µ–≥—Ä–∏—Ä—É–µ—Ç —É—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç—å –∫ —Ä–∞–∑–ª–∏—á–Ω—ã–º —Ç–∏–ø–∞–º –≤–æ–∑–º—É—â–µ–Ω–∏–π:

```
RI = w_a √ó R_a + w_n √ó R_n + w_e √ó R_e
```

- **R_a** - Adversarial —É—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç—å
- **R_n** - –®—É–º–æ–≤–∞—è —É—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç—å
- **R_e** - –£—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç—å –æ–±—ä—è—Å–Ω–µ–Ω–∏–π


#### 3. Concept-Drift Rate (CDR)

–ò–∑–º–µ—Ä—è–µ—Ç —Å–∫–æ—Ä–æ—Å—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –∫–æ–Ω—Ü–µ–ø—Ç—É–∞–ª—å–Ω—ã—Ö –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–µ–π:

```
CDR = Œª √ó KS(P_t, P_t-Œît) + (1-Œª) √ó JS(P_t, P_t-Œît)
```

- **KS** - —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ –ö–æ–ª–º–æ–≥–æ—Ä–æ–≤–∞-–°–º–∏—Ä–Ω–æ–≤–∞
- **JS** - –¥–∏–≤–µ—Ä–≥–µ–Ω—Ü–∏—è –ô–µ–Ω—Å–µ–Ω–∞-–®–µ–Ω–Ω–æ–Ω–∞


#### 4. Bias Shift Index (BSI)

–û—Ç—Å–ª–µ–∂–∏–≤–∞–µ—Ç –¥–∏–Ω–∞–º–∏–∫—É –ø—Ä–µ–¥–≤–∑—è—Ç–æ—Å—Ç–µ–π:

```

```

- **DP_Œî** - –∏–∑–º–µ–Ω–µ–Ω–∏–µ –¥–µ–º–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–æ–≥–æ –ø–∞—Ä–∏—Ç–µ—Ç–∞
- **EO_Œî** - –∏–∑–º–µ–Ω–µ–Ω–∏–µ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ —à–∞–Ω—Å–æ–≤
- **CF_Œî** - –∏–∑–º–µ–Ω–µ–Ω–∏–µ –∫–∞–ª–∏–±—Ä–æ–≤–∞–Ω–Ω–æ–π —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ—Å—Ç–∏
